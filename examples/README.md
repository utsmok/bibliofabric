# Examples

This directory contains example code showing how to build API clients using bibliofabric.

- **`simple_example.py`** — A minimal walkthrough: define a Pydantic model, implement `ResponseUnwrapper`, compose mixins into a resource client, and wire up a `BaseApiClient` subclass.

For a complete, production-quality client built on bibliofabric, see the [AIREloom](https://github.com/utsmok/aireloom) project.
